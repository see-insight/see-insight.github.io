# SEE-Insight Team Tools & Tutorials

This page lists the tools commonly used by members of the SEE-Insight Research Team, along with links to tutorials to help you get started with each one.

## Tools

- **Git** — version control: here is a list of resources you can use to learn git!:
    - [W3Schools Git Tutorial](https://www.w3schools.com/git/)
    - [Official Git Tutorial](https://git-scm.com/docs/gittutorial)
    - [Microsoft Intro to Git](https://learn.microsoft.com/en-us/training/modules/intro-to-git/)

- **Python** — primary programming language for SEE-Insight tools. Here is a list of resources to learn python:
    - [W3Schools Python Tutorial](https://www.w3schools.com/python/)
    - [Python Full Course (YouTube)](https://www.youtube.com/watch?v=XKHEtdqhLK8)

- **Jupyter** — interactive notebooks for experiments and analysis. The first link also contains anaconda python installation
    - [Anaconda Getting Started](https://www.anaconda.com/docs/getting-started/main)
    - [Jupyter Official Site](https://jupyter.org/)

- **HPCC** — MSU's High Performance Computing Center, used for large-scale experiments. Here is the documentation link:
    - [ICER Documentation](https://docs.icer.msu.edu/)


- **Scientific Image Understanding** — see [Dirk's Scientific Image Understanding tutorial](https://github.com/see-insight/Scientific_Image_Understanding).


- We also use Microsft Teams as our main source of communication along with SEE-insihgt github to store our code



## Need Help?

Reach out to your mentor or team lead if you have any questions about getting set up with these tools.
